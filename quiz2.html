Digital Systems Questions

Why is encryption important in data security?
By encrypting your information, you ensure that even if someone gets access to your computer, the files are unreadable to anyone but you.
How do usernames and passwords protect data?
They make sure that data cannot be accessed without knowledge of the username/password.
What makes a strong password? Provide an example of a very strong password.
Using a combination of letters, numbers, and symbols helps. Make it long, and hard to guess. Example: j4RRaah[nO!}
Describe a strategy on backing up data.
You should always use external backups. An example of an incremental backup is Git (with GitHub). I use Git + GitHub to back up this website. Whenever I make a change, I commit to git, and then push out to GitHub, so I can see all revisions of the website, and not lose data.
What is the difference between a full backup and an incremental backup?
Full backups save the whole project every time you make one. Incremental backups only save the changes, which is more space efficent.
Why is it important to secure data when conducting research?
Data should be secured to protect others privacy.
